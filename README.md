# Relativistic Space Explorer · 相对论太空探索者

一个基于狭义和广义相对论的星际旅行模拟器，提供 **终端 TUI** 和 **3D 浏览器游戏** 两种交互方式。

> **工作原理**：[恒星目录](js/catalog.js) (17 个真实邻近星系) → [程序化生成行星](js/planet-gen.js) → [相对论物理引擎](js/physics.js) 模拟飞船运动 → [3D 渲染](js/rendering.js) 实时呈现光行差/多普勒效应 → [HUD](js/hud.js) 显示时间膨胀/速度

## 🌌 核心特性

### 物理模拟
- **狭义相对论**：洛伦兹变换、时间膨胀、长度收缩、相对论速度叠加
- **广义相对论**：史瓦西度规下的引力时间膨胀、黑洞视界
- **相对论火箭方程**：恒本征加速度的双曲运动、多阶段任务规划（加速/滑行/减速）
- **数值积分**：RK4 变加速度轨迹

### 3D 浏览器游戏 (`game.html`)
- **真实尺度宇宙**：17 个恒星系统按真实三维坐标 1:1 分布
- **相对论光学效果**：
  - 🌀 **光行差**（Relativistic Aberration）—— 高速时星空向视野前方压缩
  - 🔵 **多普勒频移** —— 前方恒星星光偏蓝，后方偏红
  - ☀️ **集束效应**（Beaming）—— 前方亮度增强
- **3D 天体**：所有恒星和行星为 `SphereGeometry` 球体，带程序化生成纹理
- **延迟纹理生成**：靠近星系时才生成行星纹理，启动即玩
- **浮动原点**：双精度坐标下 1:1 尺度的宇宙漫游，无精度损失
- **HUD**：速度、γ 因子、时间膨胀比、最近星系信息、轨道小地图
- **引擎尾迹**：相对论速度下的可见粒子尾迹

### Python 终端版
- 完整的交互式任务规划菜单：预设/自定义任务、星图浏览、加速度对比、相对论教程
- 纯标准库，零外部依赖

## ⚛️ 相对论渲染

所有渲染计算在飞船参考系中进行：

1. **像差变换**：`n_ship = aberration(n_gal, β, γ)` 将星系方向变换到飞船系
2. **多普勒因子**：`D = 1 / [γ(1 - β·cosθ')]`
3. **颜色偏移**：`D > 1` 蓝移（蓝增强、红衰减），`D < 1` 红移
4. **集束亮度**：`亮度 ∝ D^3.5`，上限 3× 基础亮度
